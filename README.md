# Vitus Clausen - Developer Portfolio

This is a modern, responsive developer portfolio for Vitus Clausen, built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Modern Stack:** Built with Next.js App Router for optimal performance and developer experience.
- **Edgy Design:** A "digital brutalist" aesthetic with a monochrome palette and a strong accent color.
- **Responsive:** Fully responsive design that looks great on all devices, from mobile to desktop.
- **Animations:** Smooth, subtle animations using Framer Motion for an engaging user experience.
- **Theme Toggle:** Dark and light mode support.
- **Optimized for Vercel:** Ready for seamless deployment on Vercel.

## Deploying on Vercel

Deploying this portfolio to Vercel is a simple, one-click process.

1.  **Push to GitHub:** Push the code to a new repository on your GitHub account.
2.  **Import Project on Vercel:**
    *   Go to your [Vercel Dashboard](https://vercel.com/dashboard).
    *   Click "Add New..." and select "Project".
    *   Import the GitHub repository you just created.
3.  **Deploy:** Vercel will automatically detect that this is a Next.js project and configure the build settings. Simply click "Deploy".

Your portfolio will be live on a Vercel URL. You can then connect a custom domain in the project settings.
